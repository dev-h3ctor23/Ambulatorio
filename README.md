# 🏥 Práctica Global 1T: Ambulatorio

## 📝 Descripción del Proyecto
El objetivo de este proyecto es modernizar la gestión de consultas médicas en los ambulatorios. 🖥️ Este sistema web facilita la gestión de citas, consultas y medicación, mejorando la experiencia tanto para pacientes como para médicos.  

## 🌟 Características  
### 👩‍⚕️ Para Pacientes:  
- 📅 Visualización de próximas citas y detalles de las consultas pasadas.  
- 💊 Consulta de la medicación actual y su posología.  
- 🩺 Solicitud de nuevas citas con su médico de cabecera o especialistas.  

### 🧑‍⚕️ Para Médicos:  
- 🗂️ Visualización de información personal y citas programadas.  
- ✏️ Gestión de consultas:  
  - Registro de **síntomas** y **diagnóstico**.  
  - 💊 Prescripción de medicación.  
  - 📄 Generación y carga de documentos en PDF.  
  - ↪️ Derivación a especialistas.  

## 📂 Estructura del Proyecto  
El proyecto está organizado en tres páginas principales:  
1. **Paciente** 👩‍💻: Gestiona la experiencia del paciente, incluyendo citas y medicación.  
2. **Médico** 🧑‍⚕️: Proporciona herramientas para que los médicos gestionen sus consultas.  
3. **Consulta** 📋: Página detallada para registrar y gestionar información de cada consulta.  

Cada página cuenta con un archivo JavaScript asociado y está estilizada mediante CSS externo.  

## 📄 Archivos Incluidos  
- `paciente.html`, `medico.html`, `consulta.html` 👉 Archivos HTML de cada sección.  
- `paciente.js`, `medico.js`, `consulta.js` 👉 Scripts para la lógica de cada página.  
- `styles.css` 🎨 Archivo de estilos para un diseño consistente.  
- `README.md` 📘 Este archivo con información sobre el proyecto.
